# CRL2_proj

- Create venv (Python3.8)

  python3.8 -m venv myenv

- Activate venv

  source myenv/bin/activate

- Install Rasa

  pip install rasa

- Training

    rasa train

- Test chat

    rasa shell

- Integrate in telegram

    Open credential.yml, replace access_token, verify and webhook_url by your bot information
